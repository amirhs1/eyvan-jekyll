{% comment %} Read Time Include Purpose: Estimate reading time based on word count. Behavior: - Uses words-per-minute from site config (default: 200) - Accepts optional content passed into the include - Falls back to page.content when no include content is provided - Ensures a minimum display of 1 minute - Returns text only so the parent layout controls markup Dependencies: - site.words_per_minute (optional) - include.content (optional) - page.content (fallback) Usage: {% raw %}{% include read-time.html content=page.content %}{% endraw %} {% raw %}{% include read-time.html content=include.post.content %}{% endraw %} {% endcomment %} {% assign wpm = site.words_per_minute | default: 200 %} {% assign content_source = include.content | default: page.content %} {% assign words = content_source | strip_html | number_of_words %} {% if words > 0 %} {% assign read_time = words | divided_by: wpm | at_least: 1 %} {{ read_time }} min read {% endif %}
