This page verifies the functionality of the .o-hero object and its structural child zones defined in the 4-objects Sass layer.

Development note: the helper styles on this page exist only to make the layout boundaries easy to inspect.

Base object layout

Expected behavior: The hero should render as a grid object with two structural children: .o-hero__content and .o-hero__media. On smaller screens, they should stack vertically in that order.
