This page verifies the functionality of the brand classes defined in the 5-components Sass layer.

Base brand structure

Expected behavior: The brand should render as a single clickable inline-flex unit with a logo on the left and a two-line text stack on the right. The text stack should show the name and role, and the entire block should behave like one homepage link.

John Doe Applied Scientist

Long-text truncation behavior

Expected behavior: When the brand text becomes too long, both .c-brand__name and .c-brand__role should remain on a single line and truncate cleanly with ellipsis rather than wrapping. This test uses a temporary helper class so the brand fills the narrow test box and the truncation can be verified reliably.

John Doe With a Very Long Professional Display Name Applied Scientist in Computational Social Science and Digital Humanities

Compact modifier behavior

Expected behavior: Applying .c-brand--compact should force the compact treatment regardless of viewport width. The logo should shrink and the text stack should collapse away.

John Doe Applied Scientist

Responsive collapse behavior

Expected behavior: Below the component’s responsive collapse breakpoint, the default .c-brand should shift from logo + text to logo only. The text stack should visually disappear and the logo should use the collapsed size.

John Doe Applied Scientist

Resize the browser window to confirm that the default brand collapses at the responsive breakpoint without needing the compact modifier.

No-role variant

Logo-only stress case

Expected behavior: The component should remain valid and clickable even in a logo-only markup scenario. This helps verify that the logo behaves as a stable visual unit independent of the text stack.

Stable text color behavior

Expected behavior: Hover and focus should not cause the brand name or role text to shift to a different link color. The interaction cue should come from the pointer cursor and focus ring rather than a text-only color change.

John Doe Applied Scientist

Hover this brand and tab to it with the keyboard. The focus ring should appear on the whole brand link, while the text colors should remain stable.

Tip: This component is used as the reusable brand include in the site header and footer, so this test should verify both its standalone behavior and its compact header-related states.
